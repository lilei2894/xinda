## 组合搜索

### 布尔逻辑组合

```
# 日清战争相关外交文书
日清戦争 AND 外交

# 排除特定内容
満州 AND NOT 鉄道

# 多词组合
(辛亥革命 OR 民国) AND 外交

# 精确短语
"二十一条要求"
```

| 搜索条件 | 说明 |

## 高级搜索语法

### 按检索编码搜索

```
# 直接通过编码获取特定资料
refCode: A0123456789
```

### 按资料种类筛选

```
# 仅搜索公文書
关键词: 戦争 + 种类: 公文書

# 仅搜索照片
关键词: 戦争 + 种类: 写真

# 仅搜索地图
关键词: 満州 + 种类: 図画
```

### 按语言筛选

```
# 仅搜索中文资料
关键词: 战争 + 语言: 中文

# 仅搜索英文资料
关键词: war + 语言: English
```
