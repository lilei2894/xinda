# JACAR Glossary 主题分类结构

## Glossary 概述

JACAR Glossary（アジ歴グロッサリー）是一个主题分类浏览工具，帮助用户在不知道具体关键词的情况下，通过主题、机构、历史事件等分类找到相关史料。

**访问地址：**
- 英文版：https://www.jacar.go.jp/english/glossary_en/
- 日文版：https://www.jacar.go.jp/glossary/

## Glossary 搜索方法

### 方法1：浏览分类

1. 访问 Glossary 首页
2. 选择主题分类
3. 浏览子分类
4. 点击术语条目
5. 每个条目包含指向 JACAR 数据库搜索的链接

### 方法2：Glossary 内搜索

```
# 在 Glossary 搜索框中输入术语
kind: fill
ref: Glossary搜索框
text: 台湾総督府

# 执行搜索
kind: click
ref: Glossary搜索按钮
```

### 方法3：字母序索引

Glossary 提供 A-Z 字母序索引：
1. 点击字母（如 "T"）
2. 浏览该字母开头的所有术语
3. 点击术语查看详情和数据库搜索链接

## 从 Glossary 到数据库搜索

每个 Glossary 条目通常包含：
- 术语解释
- 相关机构/组织信息
- **指向 JACAR 数据库搜索的链接**（点击后自动执行预设搜索）

### 自动化流程

```
1. 导航到 Glossary
   action: navigate
   url: https://www.jacar.go.jp/english/glossary_en/

2. 选择主题分类
   kind: click
   ref: 主题分类链接（如"Taiwan"）

3. 提取所有术语条目
   evaluate: extractGlossaryEntries()

4. 对每个条目：
   a. 点击术语
   b. 提取术语信息
   c. 点击数据库搜索链接
   d. 跳转到搜索结果页
   e. 提取搜索结果

5. 保存所有结果
```

## 注意事项

1. **语言**：Glossary 英文版提供英文术语，但链接到的数据库搜索结果可能包含日文/中文内容
2. **分类层级**：Glossary 分类有 2-4 层深度，需要逐层导航
3. **搜索链接**：Glossary 条目的数据库搜索链接可能使用特定参数，点击后直接跳转到结果页
4. **更新**：Glossary 内容可能不定期更新，建议定期检查
